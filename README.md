# ПРО.АРТ — лендинг тура 2026

Быстрый статический аналог сайта концертного тура «ПРО.АРТ» (оркестр при свечах + ожившие картины).

## Стек
Чистые **HTML + CSS + JS** без фреймворков и сборки. Никакого WordPress/Elementor — отсюда скорость.

## Что внутри
- `index.html` — разметка лендинга
- `css/style.css` — стили (тёмная «свечная» тема, золото)
- `js/data.js` — данные: расписание 23 концертов, картины, программа, FAQ
- `js/main.js` — рендеринг секций, фильтр по городам, FAQ, промокоды
- `assets/` — оптимизированные медиа (видео сжаты через ffmpeg, картины WebP)

## Локальный запуск
```bash
python3 -m http.server 8765
# открыть http://127.0.0.1:8765
```

## Оптимизация
- Фоновое видео героя: 7.8 МБ → 4.0 МБ (H.264 CRF 30, без звука, 25 fps)
- Видео-отзывы: ~17 МБ → ~5 МБ (720×1280, CRF 28)
- Картины в WebP, `loading="lazy"`, видео `preload="none"`
- Нет внешних шрифтов и сторонних скриптов
